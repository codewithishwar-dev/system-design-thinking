## Redis – More Than Just a Cache

Redis may look like a simple key-value store, but under the hood it is powered by rich data structures such as:

* Strings
* Hashes
* Lists
* Sets
* Sorted Sets

Each data structure is optimized for specific use cases, enabling high performance and real-time data processing.

This is what makes Redis not just fast, but a true **data structure engine**.

### Key Takeaways

* O(1) and O(log n) operations
* In-memory processing for low latency
* Supports multiple data models
